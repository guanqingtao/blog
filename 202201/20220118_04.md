## 背景     
贡献度仅计算非 Fork 项目，Fork 项目不予计算.    
  
在某个周期内的:    
  
### 1、个人贡献度(C)  
  
行为 | 计分  
---|---  
Open Issue	| 2  
Issue & PR Comment	| 1  
Open/Merge PR	| 4(未接纳) / 5(接纳/Merge)  
PR Review	| 3  
  
### 2、项目活跃度(A)  
  
```  
A = sum(sqrt(Ci))  
```  
  
### 3、项目活跃比(R)  
  
```  
R = A/sqrt(C)  
```  
  
### 4、关注度(L)  
  
- Nf: fork数量  
- Ns: star数量  
  
```  
L = 2*Nf + Ns  
```
