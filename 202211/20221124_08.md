## DuckDB 0.6.0 CLI 支持新的结果集显示模式 duckbox, 展示头部和尾部的少量记录, 避免结果集占满屏幕

digoal                                  
                                  
### 日期                                  
2022-11-24                                 
                                  
### 标签

----                                  
                                  
## 背景                           
https://duckdb.org/2022/11/14/announcing-duckdb-060.html        
      
大结果集的查询, 只展示头部和尾部的少量记录, 避免结果集占满屏幕:     
```    
0.6.0 默认的输出格式    
    
.mode duckbox    
```    
    
如果想切回原来的模式:    
```    
.mode box    
```    
    
    
或者设置更大的显示条数    
```    
.maxrows X    
```
